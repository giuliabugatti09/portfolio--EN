## 🚀 Featured Projects

### 🏆 [Smart Insurance: AI Platform for Road Safety](https://github.com/giuliabugatti09/AI-Analytics-Platform-for-Insurance)
An award-winning end-to-end system for intelligent accident management, developed in partnership with **Sompo Seguros**.

* **Stack:** TensorFlow, XGBoost, Google Gemini API, PyDeck, Streamlit.

### [PPE Detector with Computer Vision](https://github.com/giuliabugatti09/PPE-Detector-CNN-)
Binary classification for safety helmet usage in industrial environments to enhance workplace safety.

* **Stack:** TensorFlow, Keras, Streamlit.
